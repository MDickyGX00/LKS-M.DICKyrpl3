Basic Web

Jika kamu ingin membuat sebuah web, kamu perlu mengoding. Bahasa yang sangat dasar untuk membuat web adalah bahasa HTML. HTML adalah singkatan dari HyperText MarkUp Languange. Dan jika ingin menghiasi sebuah web, kamu perlu memerlukan CSS untuk menghias. CSS adalah singkatan dari Cascading Style Sheet. Dan jika kamu ingin membuat web mu terasa responsif, kamu perlu memerlukan JavaScript.




HTML

Sederhananya, cara kerja HTML yaitu bahasa markup ini menyusun halaman website dengan tag-tag tertentu. Setiap tag punya fungsinya sendiri, sehingga Anda bisa membuat struktur dan tampilan konten yang dibutuhkan.






CSS

CSS bekerja ketika browser memuat halaman website. Untuk menerapkan pengaturan tampilan HTML yang telah ditentukan dengan kode CSS, prosesnya meliputi beberapa langkah.






JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website lebih dinamis dan interaktif. JavaScript dapat disisipkan dalam halaman web menggunakan tag SCRIPT.
